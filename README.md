# 🌍 TripAdvisor Clone

A responsive front-end **TripAdvisor Clone** built using **HTML5** and **CSS3**. This project recreates the basic layout and design of the TripAdvisor homepage with a clean and modern user interface.

## 🚀 Features

- 🧭 Navigation Menu Bar
- 🔍 Search Bar
- ⭐ Recommended Destinations Section
- 🌎 Overall Travel Categories
- 📱 Responsive Layout
- 📄 Footer Section

## 🛠️ Technologies Used

- HTML5
- CSS3
